# Datetime 日期时间选择器

用于输入日期时间，绑定的变量是JS的Date类型的变量。可以选择年月日与时分秒。

使用时，使用v-model指令绑定一个Date对象在日期选择器上，表示默认时间即可：

<s-datetime v-model="date" name="time" placeholder="input time"></s-datetime>
<script lang="ts" setup>
import {ref} from 'vue'
const date = ref(new Date());
</script>

# Datetime 属性

属性说明类型可选值默认值
modelValue/v-modelDatetimes输入框绑定的值（必须的）JS Date--
name随表单一起提交的原生的name属性string--
placeholder输入框没有值的显示内容string--
